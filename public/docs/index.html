Introduction

Base URL: http://movie-api.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Store Movie

This endpoint will create a new movie

Example request:
curl --request POST \
    "http://movie-api.test/api/movies" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "title=The Matrix" \
    --form "description=A computer hacker learns from mysterious rebels about the true nature of his reality and his role in the war against its controllers." \
    --form "length_in_seconds=7260" \
    --form "released_at=1999-03-31" \
    --form "available_until=2021-12-31" \
    --form "pictures[][title]=The Matrix" \
    --form "pictures[][description]=A computer hacker learns from mysterious rebels about the true nature of his reality and his role in the war against its controllers." \
    --form "pictures[][order]=1" \
    --form "category_ids[]=1" \
    --form "tag_ids[]=1" \
    --form "pictures[][image]=@/private/var/folders/dr/xtvt17ss22vd83pd2k507w5w0000gn/T/phpfAYyXm" 
const url = new URL(
    "http://movie-api.test/api/movies"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('title', 'The Matrix');
body.append('description', 'A computer hacker learns from mysterious rebels about the true nature of his reality and his role in the war against its controllers.');
body.append('length_in_seconds', '7260');
body.append('released_at', '1999-03-31');
body.append('available_until', '2021-12-31');
body.append('pictures[][title]', 'The Matrix');
body.append('pictures[][description]', 'A computer hacker learns from mysterious rebels about the true nature of his reality and his role in the war against its controllers.');
body.append('pictures[][order]', '1');
body.append('category_ids[]', '1');
body.append('tag_ids[]', '1');
body.append('pictures[][image]', document.querySelector('input[name="pictures[][image]"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

Request      

POST api/movies

Headers

Content-Type      

Example: multipart/form-data

Accept      

Example: application/json

Body Parameters

title   string   

The title of the movie. Must not be greater than 255 characters. Example: The Matrix

description   string   

The description of the movie. Must not be greater than 1000 characters. Example: A computer hacker learns from mysterious rebels about the true nature of his reality and his role in the war against its controllers.

length_in_seconds   number   

The length of the movie in seconds. Must be at least 0. Must not be greater than 86400. Example: 7260

released_at   string  optional  

The date when the movie was released. Format: YYYY-MM-DD. Timezone: UTC. Must be a valid date. Example: 1999-03-31

available_until   string  optional  

The date when the movie will be available until. Format: YYYY-MM-DD. Timezone: UTC. Must be a valid date. Example: 2021-12-31

pictures   object[]   
title   string   

The title of the picture. Must not be greater than 255 characters. Example: The Matrix

description   string   

The description of the picture. Must not be greater than 1000 characters. Example: A computer hacker learns from mysterious rebels about the true nature of his reality and his role in the war against its controllers.

order   number
